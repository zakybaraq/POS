# POS Application - Project Context

## Overview
Point of Sale (POS) web application for restaurant management built with Bun, Elysia, and MySQL.

## Tech Stack
- **Runtime**: Bun
- **Framework**: Elysia (Express-like)
- **Database**: MySQL with Drizzle ORM
- **Frontend**: Server-rendered HTML pages
- **Auth**: JWT with cookie-based sessions

## Project Type
- **Type**: Full-stack web application
- **Team**: Solo (1 developer)
- **Verification**: Safety first
